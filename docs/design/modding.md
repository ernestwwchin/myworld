# MyWorld RPG — Modding System

| Tier | What you can do |
|---|---|
| **Data** | Edit YAML — weapons, monsters, items, maps |
| **Tuning** | Adjust balance in `rules.yaml` |
| **Hook** | Listen to game events, trigger effects |
| **Script** | Full JS — new mechanics, UI components |
