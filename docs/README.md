## Where things go (decision rule)

When in doubt:
- "Would a player ever read this?" → `play/`
- "Would a modder?" → `modding/`
- "Would AI need it to write code?" → `engineering/`
- "Concept / intent / decision rationale?" → `design/`
- "Pure D&D 5e rule lookup?" → `ref/`
